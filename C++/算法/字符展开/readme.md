编程|100分] 字符展开
时间限制：3秒
空间限制：32768K
题目描述
输入的字符串中，含有类似于“gd-ha”或“94-73”的子串，我们就把它当作一种简写，输出时，用连续递增的字母或数字串替代其中的减号，即，将上面两个子串分别输出为“gdefgha”和“945673”。具体展开规则如下：
（1）在输入的字符串中，有且只有一个减号“-”，减号两侧的字符必须同为小写字母或同为数字，并且按照ASCII码的顺序，减号右侧的字符严格大于左侧的字符。
（2）参数p1：表示填充字符的展开方式。p1=1时，对于字母子串，填充小写字母；p1=2时，对于字母子串，填充大写字母。这两种情况下数字子串的填充方式相同。p1=3时，不论是字母子串还是数字子串，都用与要填充的字符个数相同的星号“*”来填充。
（3）参数p2：表示填充字符的重复个数。p2=k表示同一个字符要连续填充k个。例如，当p2=3时，子串“gd-ha”应扩展为“gdeeefffgggha”；子串“94-73”应扩展为“9455566673”。减号两侧的字符不变。
（4）参数p3：表示填充字符顺序/逆序标示：p3=1表示采用顺序输出，p3=2表示采用逆序输出，注意这时仍然不包括减号两端的字符。例如当p1=1、p2=2、p3=2时，子串“gd-ha”应扩展为“gdggffeeha”；子串“94-73”应扩展为“94665573”。
（5）如果减号右边的字符恰好是左边字符的后继，只删除中间的减号，例如：“d-e”应输出为“de”，“3-4”应输出为“34”。
输入描述:
第一个参数str（字符串），表示输入的ASCII字符串；

第二个参数p1（正整数，1或者2或者3），表示填充字符的展开方式；

第三个参数p2（正整数，1或者2），表示填充字符的重复个数；

第四个参数p3（正整数，1或者2），表示填充字符顺序/逆序标示；
输出描述:
若判断输入为合法的字符串，则输出处理后的字符串；

若输入不合法，则输出字符串”ERROR”；

注意：最终交付的代码中不要向控制台打印输出任何无关信息。
示例1
输入

abc3-61234 2 2 2
输出

abc3554461234
